The rules

This website is a place for students to experiment with working in git. You are expected to submit a pull request to this project.

Your change should be small and should follow these rules.

  1. You should add a <section id="P987654321"> element inside the <main> element, with the id attribute set to your student number.
  2. The element should be visible, with sensible content indicating that you have indeed been here.
  3. If you can't think of anything, just add your student number as content.
  4. You should style the element and its contents in some way by adding at least one ruleset to the CSS file like #P987654321 {}.
  5. You should not change the HTML in the rules section which contains these rules.
  6. You should not edit anyone else's content from the project.

Try to make your contribution stand out!
